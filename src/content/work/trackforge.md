A unified, high-performance computer vision tracking library implemented in Rust with Python bindings. Features state-of-the-art algorithms like ByteTrack and DeepSORT, optimized for speed and ease of use across both ecosystems.

Key features:
- ByteTrack and DeepSORT implementations
- Rust core for performance
- Python bindings for accessibility
- Real-time tracking optimization
